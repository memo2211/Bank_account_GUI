# Bank GUI Project

## Description
Bank account management system with GUI using Tkinter. Users can login, check balance, withdraw cash, change password, and perform Fawry recharges. Accounts stored in a text file as a simple database.

## Features
- User login with ID and password
- Password attempts limit and account blocking
- Check account balance
- Withdraw cash (ATM)
- Change password
- Fawry recharge service (Orange, Etisalat, Vodafone, We)

## Database
- Accounts stored in `data base.txt`
- Each account includes:
  - Name
  - Password
  - Balance
  - Status (active/blocked)
